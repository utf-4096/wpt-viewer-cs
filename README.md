# WPT Viewer

View [wpt.fyi][fyi] results in your browser with a fast and intuitive interface.

This app is in its early days - please don't expect too much out of it just yet :)

## Installation

For development:
```
pnpm i
pnpm dev
```

For deploying:
```
pnpm prod
```

[fyi]: https://wpt.fyi/results/
